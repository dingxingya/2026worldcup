# 2026worldcup
# 2026 FIFA World Cup Mini Program

> 微信小程序 - 2026美加墨世界杯赛事追踪与预测

## 项目概述

2026年FIFA世界杯（美加墨世界杯）专属微信小程序，覆盖赛事信息浏览、小组积分实时追踪、淘汰赛对阵、用户预测等核心功能。

- **赛制**: 48支球队，12个小组，104场比赛
- **时间**: 2026年6月12日 - 7月20日
- **举办地**: 美国 · 加拿大 · 墨西哥
- **AppID**: `wx4e34d9981dabf015`

## 淘汰赛对阵规则

### 出线规则

1. 每组前2名（24队）直接晋级32强
2. 12个小组第3名中取成绩最好的8个晋级
3. 共32队进入淘汰赛

### 第3名排名依据

积分 > 净胜球 > 进球数 > 公平竞赛分 > 抽签

### 淘汰赛阶段

1. **1/16决赛** (16场): 6月29日 - 7月4日
2. **1/8决赛** (8场): 7月5日 - 7月8日
3. **1/4决赛** (4场): 7月10日 - 7月12日
4. **半决赛** (2场): 7月15日 - 7月16日
5. **三四名决赛** (1场): 7月19日
6. **决赛** (1场): 7月20日 @ 大都会人寿体育场，纽约

## 核心业务逻辑

### 积分计算 (utils/standings.js)

```
胜 = 3分, 平 = 1分, 负 = 0分
排名: 积分 > 净胜球 > 进球数 > 相互战绩
```

### 出线判断 (utils/qualification.js)

- `getRankedThirdPlaces()`: 获取所有小组第3名并排序
- `getQualifiedThirdPlaces()`: 判断哪些第3名出线（取前8）
- `getQualifiedTeams()`: 获取完整32强名单
- `checkTeamQualification()`: 判断单支球队的出线状态

### 预测数据存储

```javascript
// 小组出线预测
wx.getStorageSync('predictions')
// { "A": { "first": "MEX", "second": "KOR" }, ... }

// 淘汰赛/冠军预测
wx.getStorageSync('bracketData')
// { "R32M1": "A1", "Final": "ARG", ... }
```
